Library Manager Panel

Library manager panel allow you to manage the library's collection. You can add new books, view library collection and change books amount. Each operation can be executed by user-friendly interface. List of available operations:

  1. View books

    2. You can view our entire available collection of books. You can check total book amount and currently available book amount.

  2. Change book amount

    3. You can modify amount of each book in library collection.

  3. Add books

    4. There you can see add new books to library collection. You have to specify title, publication year, author name and genre.
