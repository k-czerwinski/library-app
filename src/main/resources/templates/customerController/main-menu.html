Library Customer Panel

Library customer panel allow you to borrow and search for book. Furthermore, you can return books which you have already borrowed. Each operation can be executed by user-friendly interface. List of available operations:

  1. View books

    2. You can view our entire available collection of books. When you click on column header entire table will be sorted by that factor. You can add there book to cart.

  2. Search for a book

    3. You can search for book which you are interested in. Panel will show you available options from which you can choose book to add cart.

  3. Return books

    4. There you can see list of books which you have already borrowed and return its.

  4. Cart interface

    5. There are stored books which you added to cart. By cart interface you can remove books from cart or confirm order.
